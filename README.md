# ai_tools_exploration
Explore few AI tools for Excel, Powerpoint, etc.

## Project1: Data Entry Automation
In this project, use the Optical Character Regonition (OCR) capabilities of Excel, to get the data which is in image format to excel. This would help in minimizing the manual efforts in data entry and also minimizing human errors.

##### Input File Used: image_to_data_entry_in_excel.jpeg
##### Output: data_entry_automation.xlsx

###### Steps:
1. Open an empty excel file
2. Go to Insert -> From Picture
3. Select the image in your machine
4. Let the data from picture be extracted
5. After the data is extracted, you can review the data and then insert the data on the reuqired cell by selecting the cell

## Project2: AI formula generation in excel
In this project, use generative ai tools like copilot, chatgpt and generate the formula. We can use the copilot web version or copilot within excel too.

##### Input File Used: excel_formula1_ai_input.xlsx
##### Output: excel_formula1_ai_output.xlsx

###### Steps:
1. Open copilot web version https://copilot.microsoft.com/chats/ or chatgpt https://chatgpt.com/ or use the copilot already present in the excel
2. Give a very good prompt as given in the excel file
3. Get the formula and use it in the excel
